# ContextGraph 0.4.0

## Summary

ContextGraph `0.4.0` turns the repo into a more credible developer beta for teams building multi-agent support and research systems.

This release does three important things:

- sharpens the product around governed shared memory and discovery
- adds visible lifecycle and sentinel governance features
- improves the onboarding and demo path so teams can feel value quickly

## Highlights

### Discovery profiles

- separate profile-level discoverability model
- discover/search/filter API
- discover dashboard page
- visible agent profile, activity, and trust views

### Governance and lifecycle

- built-in sentinel agents
- stored verdict visibility
- trust summaries with status and verdict counts
- suspend/reactivate/delete agent flows

### Beta launch hardening

- aligned versioning to `0.4.0`
- clearer README positioning
- runnable beta quickstart
- support and research workflow examples
- production readiness guide
- design-partner launch material

## Upgrade Notes

- profile discoverability is now separate from memory defaults
- discovery docs and plans were rewritten to reflect the shipped model
- the old one-click runtime direction was reframed as an external orchestrator integration, not an in-repo runtime
- the large audit control-plane proposal is now marked as roadmap-only

## Known Limitations

- dashboard auth is still agent-based, not user-account based
- no hosted control plane ships with this repo
- no export API or backup automation layer exists inside the app
- no native runtime hosting exists in this repo
- broader audit alerts/reports control plane is not implemented
