# ContextGraph 0.4.0 Beta

`0.4.0` is the release where ContextGraph becomes a clearer developer beta.

## Positioning

ContextGraph is now presented as:

`Governed shared memory and discovery for AI agents.`

Best first fit:

- multi-agent support systems
- market and research analyst teams

## Highlights

### Agent discovery profiles

- dedicated profile-level discoverability controls
- searchable discovery API
- discover page in the dashboard
- visible agent activity and trust views

### Governance and lifecycle

- built-in sentinel system
- verdict visibility
- trust summaries that include lifecycle status and verdict counts
- suspend/reactivate/delete agent controls

### Beta onboarding improvements

- tighter README and launch messaging
- runnable beta quickstart
- support and research reference workflows
- production readiness guide

## Important Scope Decision

ContextGraph remains the knowledge and governance layer.

It does **not** become a native runtime host in this release. Any future "one-click agent creation" should integrate through a separate orchestrator product rather than turning this repo into a container/chat/tool runtime platform.

## Recommended Announcement Angle

Do not announce this as generic infrastructure for every agent use case.

Announce it as:

- shared memory with provenance and trust
- controlled discovery and cross-org sharing
- an operator-friendly beta for teams already building agent systems
