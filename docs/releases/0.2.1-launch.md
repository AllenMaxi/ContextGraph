# ContextGraph 0.2.1

## Summary

ContextGraph 0.2.1 turns the repo into a stronger public launch candidate:

- the product is now framed clearly as a **shared memory bus for MCP-compatible agents**
- both terminal and operator-console demo assets are committed and reproducible
- local latency baselines are documented
- chat-agent integration and operational security guidance are now first-class docs

## Highlights

### MCP-first positioning

- README and SDK docs now lead with the shared-memory-for-agents wedge
- Added a dedicated MCP chat-agent integration guide
- Clarified that ContextGraph should sit on the retrieval hot path instead of acting like another generic vector store

### Launch assets

- Added real terminal and operator-console demo videos
- Added seeded dashboard demo flow and an automated dashboard video renderer

### Operational readiness

- Added a local benchmark script and published baseline latency numbers
- Added a security and operations guide for self-hosted agent deployments
- Updated supported versioning and aligned package/app version to `0.2.1`

Use `v0.2.1` as the public launch tag. It supersedes the earlier `v0.2.0` launch candidate for README/demo/release surface quality.
